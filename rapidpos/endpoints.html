MCP Tool Reference

Endpoints.

All tools exposed by the RapidPOS MCP Server. JSON-RPC 2.0. Read-only tools return immediately; transactional tools require customer consent scope.

POST https://<tenant>.rapidpos.growdirect.io/mcp jsonrpc: 2.0 · auth: bearer
ReadReturns immediately. No side effects.
TransactionalRequires customer consent scope.
KnowledgeDomain vault. Does not hit Counterpoint.

01

inventory_lookup

Read

Real-time stock query across one or more store locations.

Input
sku string REQUIRED
Counterpoint SKU identifier.
store_ids string[]
Array of store IDs. Omit for all stores.
include_holds bool
Include current holds in availability calc. Default false.
Output
sku string
stores[] array
Per-store: id, on_hand, on_hold, available.
cache_age_seconds int
How stale the cached value is.

Note. Cached via Redis. Cache TTL 60s in Phase 1; event-invalidated in Phase 2.

Request 
{
  "jsonrpc": "2.0",
  "method": "tools/call",
  "params": {
    "name": "inventory_lookup",
    "arguments": {
      "sku": "AGAVE-BLU-5G",
      "store_ids": ["PAS", "LAG"],
      "include_holds": true
    }
  },
  "id": 1
}
Response 
{
  "sku": "AGAVE-BLU-5G",
  "stores": [
    { "id": "PAS",
      "on_hand": 4,
      "on_hold": 1,
      "available": 3 },
    { "id": "LAG",
      "on_hand": 7,
      "available": 7 }
  ],
  "cache_age_seconds": 18
}
02

customer_profile

Read consent: customer:read

Retrieve customer record by email or loyalty ID.

Input
lookup_type enum REQUIRED
One of email or loyalty_id.
value string REQUIRED
The lookup value matching the chosen type.
Output
customer_id string
name string
loyalty_tier string
points_balance int
purchase_history_summary object
garden_record object
Vertical-specific. Plants, planting dates, zone.

Note. Requires customer:read consent scope on the active session.

Request 
{
  "name": "customer_profile",
  "arguments": {
    "lookup_type": "email",
    "value": "j.rivera@example.com"
  }
}
Response 
{
  "customer_id": "C-018241",
  "name": "Jasmine Rivera",
  "loyalty_tier": "Cultivator",
  "points_balance": 2840,
  "garden_record": {
    "sunset_zone": 22,
    "plants": 14,
    "last_planted": "2025-09-18"
  }
}

04

place_hold

Transactional consent: transactions:write

Reserve an item at a specific store for a customer.

Input
sku string REQUIRED
store_id string REQUIRED
customer_id string REQUIRED
quantity int
Default 1.
hold_duration_hours int
Default 48.
Output
hold_id string
expires_at datetime
confirmation string
Human-readable confirmation code for in-store pickup.

Note. Requires transactions:write consent scope. Creates a Counterpoint hold record. Idempotent on (customer_id, sku, store_id) within 60s.

Request 
{
  "name": "place_hold",
  "arguments": {
    "sku": "AGAVE-BLU-5G",
    "store_id": "PAS",
    "customer_id": "C-018241",
    "quantity": 1
  }
}
Response 
{
  "hold_id": "H-2026-04-26-01142",
  "expires_at": "2026-04-28T19:00:00Z",
  "confirmation": "PAS-1142"
}
05

price_lookup

Read

Current POS price for a SKU, including active promotions.

Input
sku string REQUIRED
store_id string REQUIRED
customer_id string
Optional. If provided, applies loyalty pricing.
Output
regular_price decimal
sale_price decimal
promotion_name string
loyalty_price decimal
Request 
{
  "name": "price_lookup",
  "arguments": {
    "sku": "AGAVE-BLU-5G",
    "store_id": "PAS",
    "customer_id": "C-018241"
  }
}
Response 
{
  "regular_price": 39.99,
  "sale_price": 34.99,
  "promotion_name": "Spring Native Plants",
  "loyalty_price": 31.49
}
06

plant_diagnostic

Knowledge

Symptom-to-treatment lookup from the domain knowledge vault.

Input
symptoms string[] REQUIRED
Free-text or coded symptom labels.
plant_type string
Optional. Species or common name.
sunset_zone int
Optional. Sunset climate zone.
Output
diagnosis string
confidence float
treatment_steps string[]
recommended_products array
SKUs from current inventory matching the treatment.
care_notes string

Note. Knowledge layer tool. Does not hit Counterpoint — returns from the domain vault. Recommended SKUs are reconciled against live inventory before being returned.

Request 
{
  "name": "plant_diagnostic",
  "arguments": {
    "symptoms": [
      "yellowing leaves",
      "powdery white spots"
    ],
    "plant_type": "rose",
    "sunset_zone": 22
  }
}
Response 
{
  "diagnosis": "Powdery mildew",
  "confidence": 0.91,
  "treatment_steps": [
    "Improve air circulation",
    "Apply potassium bicarbonate spray"
  ],
  "recommended_products": [
    { "sku": "FUNG-PB-32",
      "name": "Bonide Remedy 32oz" }
  ]
}
